* Overview

Pure function that filters RSS items by keywords, compiled from Rust to WASM.

Given a JSON input containing RSS items and a list of keywords, this skill
returns only the items whose title or description matches at least one keyword.

* Files

- =skill.toml= — Capability manifest
- =src/lib.rs= — Rust source (compiled to WASM)
- =Cargo.toml= — Rust build configuration
- =README.org= — This file

* Building from Source

#+begin_src bash
# From this directory (examples/skills/rss-filter/)
cargo build --target wasm32-unknown-unknown --release
#+end_src

The compiled WASM binary will be at:

=target/wasm32-unknown-unknown/release/rss_filter.wasm=

** Prerequisites

#+begin_src bash
# Install the WASM target if you haven't already
rustup target add wasm32-unknown-unknown
#+end_src
