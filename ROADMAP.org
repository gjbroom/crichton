* Overview

Crichton is a secure background AI agent daemon built in SBCL Common Lisp,
designed with security-first principles derived from a real-world audit of
OpenClaw. This roadmap tracks completed work and future development.

See [[file:AGENT_SPEC.md][AGENT_SPEC.md]] for the full design specification.

* Explicitly NOT Planned for v0.1-v0.2

The following were considered but explicitly deferred or rejected:

- Public marketplace for skills (security risk - requires extensive review process)
- Auto-install of skills (security risk - manual review required)
- LAN/remote access mode (security complexity - local-only for now)
- Arbitrary-language skill runtimes (stick to WASM for security)
- Mobile apps (desktop/server focus)
- Shell-based skills (security risk - WASM only)

** Architecture Highlights

- Language: Common Lisp (SBCL)
- Microkernel: Daemon + separate skill runner
- Sandboxing: WASM (wasmtime) with capability manifests
- Encryption: age (credentials & sessions)
- Signing: Ed25519 (ironclad)
- Logging: Structured JSON, 0600 permissions
- Management: systemd user service
